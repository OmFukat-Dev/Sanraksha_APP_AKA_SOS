# Project Summary
**SOSApp (Sanraksha Alert)** is a professional Android emergency safety application designed to provide quick SOS alerts in life-threatening situations. This local-first app operates without server dependencies, utilizing Room Database for user and contact management. Key features include secure logins, multiple SOS triggers (like shake, voice, and sound detection), and automatic emergency responses via SMS or calls with GPS location sharing. The app is designed with a Material3 theme, ensuring a modern and user-friendly experience while prioritizing user privacy by storing all data locally.

# Project Module Description
- **Authentication**: Unique ID login, account creation, and forgot ID functionality.
- **SOS Triggers**: Manual SOS button, shake detection, voice recognition, and sound detection.
- **Emergency Response**: Automatic SMS with location and calling the first emergency contact.
- **Contacts Management**: Add, edit, and delete emergency contacts with swipe gestures.
- **User Profile**: Manage user details like name, phone, email, blood group, and age.
- **Settings**: Toggle safety modes, adjust shake sensitivity, and switch between dark/light themes.
- **Background Monitoring**: Foreground service for continuous SOS monitoring.
- **Security/Privacy**: Local database with encrypted sensitive data.

# File Description Inventory
- **build.gradle**: Project-level build configuration.
- **settings.gradle**: Project settings and module inclusion.
- **app/build.gradle**: App-level build configuration.
- **AndroidManifest.xml**: Application permissions and activity declarations.
- **User.kt, Contact.kt**: Database entities.
- **UserDao.kt, ContactDao.kt**: Data access objects for database operations.
- **AppDatabase.kt**: Singleton for Room database instance.
- **PrefManager.kt**: SharedPreferences manager for storing user settings.
- **LocationHelper.kt**: Utility for fetching device location.
- **SMSHelper.kt**: Utility for sending SMS messages.
- **ShakeDetector.kt, VoiceDetector.kt, SoundDetector.kt**: Utilities for detecting SOS triggers.
- **SOSTriggerManager.kt**: Manages the SOS triggering process.
- **SOSMonitoringService.kt**: Foreground service for monitoring SOS triggers.
- **ContactsAdapter.kt**: RecyclerView adapter for displaying contacts.

# Usage
1. **Install Dependencies**: Use Gradle to sync and install project dependencies.
2. **Build the Project**: Compile the project in Android Studio.
3. **Run the Application**: Deploy the application on an Android device or emulator.
4. **Add Siren Sound**: Place `siren_sound.mp3` in `app/src/main/res/raw/` before building.
